# ToDo-list-python
A simple Python To-Do List with deadlines and priority tracking
## Features:-
 Add tasks with name, priority, and deadline
- View all tasks in a clean format
-  See how many days are left for each task
-  Get alerted if a task is overdue
-  Mark tasks as done
-  Delete tasks
#📌 Deadline Format:-
Always enter the deadline in this format: `DD-MM-YYYY`
Built With Python 3
- `datetime` module (built-in, no installs needed)
